## docker常用命令

```
docker start [name] #启动docker镜像
docker stop [name] #停止docker镜像
docker restart [name] #重启docker镜像
docker logs -f [name] #输出docker镜像日志
docker ps -a #显示已安装的所有docker镜像
docker exec -it [name] #进入docker镜像内部
```
